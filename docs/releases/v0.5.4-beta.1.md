# JAVM v0.5.4-beta.1

> `v0.5.3` → `v0.5.4-beta.1`

## 新增

- **发现页 & 演员中心**：按演员、片商、系列、导演、分类多维度浏览；支持演员档案与作品全集在线抓取、收藏、在线搜索，以及演员多名字/别名自动合并（同一演员的不同译名归并为一人）。
- **有码 / 无码分轨**：自动识别有码无码并分别路由数据源，支持按类型筛选与「一键无码」模式，新增原生无码源。
- **并行视频源查找**：多个资源站同时查找，每个源一行实时进度、各自给出正片链接列表；自动屏蔽广告/片段，404 页即时结束、不再干等。
- **多源元数据融合**：集成 MetaTube 聚合源（缺失/损坏时自动下载更新），多源并发抓取并按字段融合出更完整的刮削结果。
- **封面与画廊**：新增封面横屏/竖屏设置、瀑布流等高画廊模式，卡片大小可拖拽调节。
- **视频去重批量删除**：打开即按时长完整度、分辨率、体积智能勾选低质重复项、保留最优，一键批量删除。
- **刮削信息独立存储**：支持 NFO 与图集和视频文件分离存放，图集标准化（poster/fanart/thumb）。
- **预发布更新通道**：应用内更新支持 Beta / RC 通道。
- **番号搜索增强**：识别覆盖更广，无连字符也能搜（如 `ssis666` 与 `ssis-666` 结果一致）。

## 优化

- 刮削更稳定、更不易被目标站拦截（限速、退避、UA / 代理 / 镜像轮换）。
- 详情页融合刮削速度提升。
- 自动过滤过小的预览图；封面跨源择优并校验有效性，杜绝裂图。

## 修复

- 修复并行查找中 404 页面不自动关闭、以及手动关闭查找窗口后卡住不推进下一个源的问题。
- 修复封面在部分情况下裂图、占位图未回退的问题。
- 修复演员仅改大小写/空格被误判丢失、全集按别名合并时漏作品等问题。
- 修复若干弹窗的无障碍告警。
